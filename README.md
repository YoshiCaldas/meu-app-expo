# FitLife - Personal Trainer Digital

Aplicativo mobile desenvolvido com React Native e Expo, criado para auxiliar usuários no acompanhamento da rotina fitness, oferecendo treinos personalizados, controle de progresso físico, hidratação diária e planejamento alimentar.

## Objetivo

O FitLife tem como objetivo funcionar como um personal trainer digital, permitindo que o usuário monte treinos personalizados de acordo com seu peso, altura, objetivo físico e disponibilidade semanal. Além disso, o aplicativo auxilia no acompanhamento da evolução corporal e hábitos saudáveis.

## Funcionalidades

* Cadastro e login de usuários
* Montagem automática de treinos personalizados
* Seleção dos dias da semana para treinamento
* Escolha dos grupos musculares para cada dia
* Divisões de treino inspiradas nos métodos mais utilizados na musculação
* Cálculo automático do IMC
* Controle de peso corporal e evolução semanal
* Histórico de progresso físico
* Controle de ingestão diária de água
* Definição de meta personalizada de hidratação
* Seleção de dietas de acordo com o objetivo do usuário
* Atualização automática dos dados na tela inicial
* Interface moderna e responsiva
* Compatibilidade com Android, iOS e Web

## Como Instalar

git clone https://github.com/YoshiCaldas/meu-app-expo.git

cd meu-app-expo

npm install

## Como Executar

npx expo start

Após executar o comando, utilize o aplicativo Expo Go ou um emulador Android/iOS para visualizar o projeto.

## Como Gerar o APK

npx eas build:configure

npx eas build -p android --profile preview

## Principais Recursos do Aplicativo

### Treinos Personalizados

O usuário informa peso, altura e objetivo. Com base nessas informações, o aplicativo gera treinos utilizando exercícios populares da musculação e diferentes divisões de treino.

### Cronograma Semanal

Permite selecionar os dias da semana e definir qual grupo muscular será treinado em cada dia.

### Evolução Física

Registro semanal de peso para acompanhar ganhos ou perdas ao longo do tempo.

### Controle de Hidratação

Definição de meta diária de água e acompanhamento do consumo durante o dia.

### Dietas

Sugestões de dietas voltadas para hipertrofia ou emagrecimento, de acordo com o objetivo selecionado.

### IMC Automático

Cálculo instantâneo do Índice de Massa Corporal utilizando peso e altura cadastrados.
