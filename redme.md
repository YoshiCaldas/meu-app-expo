Aplicativo mobile desenvolvido com React Native e Expo, criado para demonstrar o desenvolvimento de aplicações multiplataforma de forma simples e eficiente.
Objetivo
O projeto tem como objetivo servir como base para estudos e desenvolvimento de aplicativos mobile utilizando React Native e Expo. Ele demonstra a estruturação de um projeto moderno, permitindo a execução em dispositivos Android, iOS e Web.
Funcionalidades
Interface mobile responsiva
Navegação entre telas
Compatibilidade com Android, iOS e Web
Atualização em tempo real durante o desenvolvimento
Estrutura organizada para expansão de funcionalidades
Tecnologias utilizadas
React Native
Expo
JavaScript
Node.js
npm
Como instalar
git clone https://github.com/YoshiCaldas/meu-app-expo.git
cd meu-app-expo
npm install
Como executar
npx expo start
Após executar o comando, utilize o aplicativo Expo Go ou um emulador Android/iOS para visualizar o projeto.
Como gerar o APK
npx eas build:configure
npx eas build -p android --profile preview
